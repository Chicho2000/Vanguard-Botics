funcion de logout:

habria que borrar el token que le dimnos al usuario (en el front) y en el back 

router.get para tener una ruta para verificar el token (para ver si e usuario sigue loggeado, si el token sigue siendo valido, etc.). aca hago una request GET al back y me dice si sigue loggeado o no. middleware (se le importa una libreria) que verifica que el token sea valido (es una funcion que se ejecuta antes de que se ejecute la funcion que queremos ejecutar). devolvemos token valido o no valido

middleware: es una funcion que generalmente toma 3 parametros (req, res, next). se toma los headers authorization y devolvemos 401 si no esta el token. si esta el token, verificamos que sea valido y si es valido llamamos a la funcion que queremos ejecutar (next).
